# Ultra Portfolio

Live Version: https://ghala.free.je/?i=1  
Click to view the complete portfolio and explore the work behind its design.

A modern personal portfolio built using HTML and CSS, featuring a clean layout, structured sections, and smooth visual styling. The project is deployed on InfinityFree as a live website.

## Technologies Used

- HTML5 for page structure  
- CSS3 for layout, styling, gradients, animations, and visual effects  
- InfinityFree for hosting and deploying the website on a live server  

## Project Description

The portfolio includes several main sections such as About, Skills, Projects, and Contact.  
The design focuses on clarity, smooth visual elements, and organized content.  
Styling was implemented using CSS only, with an emphasis on gradients, shadows, and subtle animations to create a modern interface without relying on external frameworks.

## Development Process

### Structure
The page layout was built using HTML, dividing the content into clear sections to ensure readability and easy navigation.

### Styling
CSS was used to define the visual identity of the site, including background gradients, glass‑style containers, hover effects, and animation transitions.

### Deployment
The final version of the project was uploaded to InfinityFree, where the files were hosted and linked to a public domain to make the website accessible online.
